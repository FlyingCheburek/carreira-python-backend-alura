# Curso: Carreira Back-end com Python da ALURA
Nesse curso, pude desenvolver todas as habilidades, do básico ao avançado, para trabalhar com Back-end de aplicações web utilizando a linguagem Python, com foco no framework Django e Flask. 

## Habilidades:
- Fundamentos computacionais
- Fundamentos básicos Python
- Fundamentos de web
- Git e Github
- Metodologias ágeis e DevOps
- REST
- Microsserviços
- Integração assíncrona
- Flask
- Django
- Kubernetes
- Kafka
